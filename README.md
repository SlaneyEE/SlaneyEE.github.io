## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Biome (linting & formatting)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Linting & Formatting

```bash
# Check for issues
npm run lint

# Fix issues
npm run lint:fix

# Format code
npm run format
```
